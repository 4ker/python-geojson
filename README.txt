GeoJSON: encode/decode geodata
==============================

This package contains 3 things:

* The normative definition of a Python Geo Interface, which is
  currently under development at: 
  
  http://trac.gispython.org/projects/PCL/wiki/PythonGeoInterface.

* simplejson-based classes and functions for encoding and decoding a flavor of
  Geo-JSON.

* A working example of a class that satisfies the Python Geo Interface.
